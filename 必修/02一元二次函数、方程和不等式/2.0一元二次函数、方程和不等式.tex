\chapter{不等式}

本章讨论不等式。不等式常用于讨论函数最大值最小值，在工程实践中也有现实意义。

本章要点：
\begin{itemize}
    \item 不等式的性质。
    \item 四类均值及其不等式。
    \item 基本不等式和二次函数。
\end{itemize}
