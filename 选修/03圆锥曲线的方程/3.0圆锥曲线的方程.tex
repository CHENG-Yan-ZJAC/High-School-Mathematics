\chapter{圆锥曲线的方程}

本章介绍3类二次曲线——椭圆、双曲线和抛物线。它们都可以通过圆锥获得，所以通称圆锥曲线，事实上，圆也是一类圆锥曲线。

本章要点：
\begin{itemize}
    \item 椭圆。
    \item 双曲线。
    \item 抛物线。
\end{itemize}
